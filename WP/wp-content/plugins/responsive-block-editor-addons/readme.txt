== Description ==

Build professional and elegant websites using the Gutenberg editor. With the enhanced blocks and widgets in the Responsive Gutenberg Blocks Library, you can design high-converting landing pages. 

The library gives you the capability to create layouts and pages without writing a single line of code. The blocks library has 30+ unique blocks having customizable features for each one.

[Take a look at the demo blocks](https://cyberchimps.com/blocks) to understand their features.

This makes the process of designing websites and writing content simpler and flexible. The Responsive Blocks plugin is lightweight and it exceeds the limitations of the inbuilt Gutenberg page builder to make the process of building websites enjoyable even for non-coders.

== FREQUENTLY ASKED QUESTIONS ==
You can reach out to our dedicated support team if you have queries. We’d be happy to sort out all questions related to the plugin.


= 1. Is the Responsive Gutenberg Blocks Library plugin usable on any website? =
These addons are designed to work on any niche website and are highly customizable to create beautiful landing pages and posts in Gutenberg.

= 2. Which theme is compatible with this Responsive Gutenberg Blocks Library? =
This plugin works with any WordPress theme but it is advisable to use it with the Responsive theme by CyberChimps.

= 3. Will this plugin reduce the speed of my website? =
Definitely not! These add-ons have been designed keeping in mind the necessity of responsive web design practices. So users don’t have to worry about the page load time getting affected. These addons will work on mobiles and desktops without reducing the page speed.

= 4. Can I use custom fonts and my own icons? =
Yes. You can use custom fonts and add your own icons while designing your pages.

= 5. Can I use the Responsive Gutenberg Blocks Library plugin by keeping the other page builder plugins activated? =
Yes, You can keep this plugin activated along with any other page builder like Elementor.


== Installation ==

This section describes how to install the plugin and get it working.

1. In the WordPress dashboard of your website, click on Plugins
2. Click “Add New”
3. Upload the Zip file of the plugin
3. Activate the plugin
